Плата KVP_03v2.
Сделана в OrCAD 16.6.

!!! ЦС на этих платах использоваеть не рекомендуется, поэтому не запаивается.
Предназначена для использования в АВАНТ К400 и АВАНТ Р400М вместе с сэндвич-
панелями.

Изменения:
На разъеме перекинуты сигналы Rx/Tx для локальной сети и телемеханики.
Резисторы подтяжек интерфейсов заменены на 1206.
У выходных сигналов интерфейсов изменены подтяжки, т.е. Y и Z подтянуты к другим 
уровням.

Основной вариант:
используется для Р400М
отсутствуют: цифровая ретрансляция, выходы ком.ПРД/ПРМ, вход ПУСК

Вариант К400:
используется для К400 и РЗСК
плата запаивается полностью

Входы управления могут быть 24В, 110В или 220В.
